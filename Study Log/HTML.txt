기본적으로 express는 views라는 파일안에서 view 파일들을 찾는다(html같은거)
-> 현재 작업 위치(Current Working Directory : cwd) 기준으로 view임!
-> 기본적으로 package.json이 우리의 서버 기동 장치이기 때문에... 일반적으로 최상위가 cwd가 됨
-> 변경이 필요하면... app.set("views", process.cwd() + "/src/views");

-> res.render("home");
